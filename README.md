# PTA適正化推進委員会ホームページ（v2）

本リポジトリは、PTA適正化推進委員会の公式ホームページの初期構築版です。
静的なHTML/CSS/JavaScriptのみで構成されており、フレームワークやビルドツールには依存していません。

GitHub Pagesでの公開を想定しています。

## 主要ファイル

| ファイル | 内容 |
| --- | --- |
| `index.html` | トップページ本体。PTAと学校の境界に関する論点を本文中心で解説する。 |
| `assets/css/style.css` | サイト全体のスタイル。白地・濃紺基調で、資料サイトとして読みやすいレイアウトを構成する。 |
| `assets/js/main.js` | 最小限のJavaScript（スマホ用ナビゲーションの開閉、フッターの年表示）。 |
| `404.html` | 存在しないページにアクセスした際の案内ページ。 |
| `robots.txt` | クローラー向けの最小設定。 |
| `sitemap.xml` | サイトマップ（公開URL確定後に要確認）。 |

## 公開URLについて

現時点では `https://ptaorg.com/` を想定URLとして `robots.txt` および `sitemap.xml` に記載しています。
公開URLが確定した際は、`sitemap.xml` と `robots.txt` の記載内容を必ず確認し、必要に応じて修正してください。

## 今後追加すべきページ候補

現時点ではトップページのみの最小構成です。今後、以下のようなページの追加を検討してください。

- 学校とPTAの境界に関する論点整理の詳細ページ
- 自治体・教育委員会向けの点検の手引きページ
- 保護者向けのよくある質問（FAQ）ページ
- 関連する制度・通知等の資料一覧ページ（内容が確認できたもののみ掲載）

## 運用上の注意事項

このサイトは、カードやリンクを並べただけのポータルサイトではなく、**本文で論理を読ませる資料サイト**として運用することを前提としています。
ページを追加する際も、以下の方針を維持してください。

- 本文による説明を主とし、カード・タグ・リンク一覧は本文の補助にとどめる。
- 「詳しくはこちら」のような誘導だけで本文を終えない。
- 確認していない法令・判例・URL・資料へのリンクを掲載しない。
- 大量のページやカードを一度に増やさない。
